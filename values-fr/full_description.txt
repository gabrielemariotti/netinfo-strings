<b>Net info</b> est une petite appli pour Android Wear.
<b>OK Google... ouvrir net info.</b>
Maintenant vous pouvez vérifier les informations réseau de votre mobile sans avoir besoin de sortir votre téléphone de votre poche.

<b>Fonctionnalités</b>
• état du wifi : connecté/non connecté
• nom du réseau wifi (SSID)
• vitesse du wifi
• État du mobile : connecté/non connecté
• Opérateur mobile
• Type de connexion mobile

<b>Veuillez noter : un appareil Android Wear est requis !</b>
