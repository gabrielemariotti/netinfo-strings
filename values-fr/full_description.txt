<b>Net info</b> est une application légère pour Android Wear.
<b>OK Google... lance net info.</b>
Vous pourrez alors vérifier les informations détaillées du réseau sans sortir votre smartphone de votre poche.

<b>Fonctionnalités</b>
• Statut du wifi: connecté/non connecté
• Nom du réseau wifi (SSID)
• Débit wifi
• Statut du réseau mobile: connecté/non connecté
• Opérateur mobile
• Mode de connection mobile

<b>Veuillez noter qu'un appareil Android Wear est requis!</b>
